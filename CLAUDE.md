# OHM Cycles Website

## Overview
Shopify-based e-commerce website for OHM Cycles. This repo is primarily used for issue tracking and build prioritization via GitHub Issues and Projects.

## Tech Stack
- **Platform**: Shopify
- **Theme**: TBD
- **Tracking**: GitHub Issues + GitHub Projects

## Common Tasks

```bash
# Create a new issue
gh issue create --title "Issue title" --label "priority: high,type: content"

# List open issues by priority
gh issue list --label "priority: critical"
gh issue list --label "priority: high"

# View project board
gh project view --owner mdevisser
```
