# OHM Cycles Website

Shopify website build tracking and issue management for [OHM Cycles](https://ohmcycles.com).

## Issue Labels

### Priority
- `priority: critical` — Blocking launch
- `priority: high` — Must have for launch
- `priority: medium` — Should have
- `priority: low` — Nice to have / post-launch

### Type
- `type: content` — Copy, images, media
- `type: design` — Layout, styling, branding
- `type: functionality` — Features, integrations, apps
- `type: bug` — Something broken
- `type: seo` — SEO, metadata, analytics
- `type: advertising` — Ads, campaigns, media buying

### Area
- `area: homepage` — Homepage
- `area: product` — Product pages
- `area: collection` — Collection pages
- `area: cart-checkout` — Cart & checkout
- `area: navigation` — Menus, header, footer
- `area: global` — Site-wide / theme-level
- `area: google-ads` — Google Ads campaigns & tracking
- `area: meta-ads` — Meta/Facebook/Instagram ads
- `area: email` — Email marketing (Klaviyo)

### Phase
- `phase: 1-foundation` — Brand, nav, metafields setup
- `phase: 2-homepage` — Homepage section optimization
- `phase: 3-products` — Product page updates
- `phase: 4-collections` — Collection page updates
- `phase: 5-pages` — Supporting pages
- `phase: 6-seo-tech` — SEO and technical optimization
- `phase: 7-advertising` — Advertising setup & optimization

### Status
- `status: blocked` — Waiting on something external
- `status: needs-info` — Needs more detail before starting
